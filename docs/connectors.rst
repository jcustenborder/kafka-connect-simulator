===================
Simulator Connector
===================

The Simulator Connector provides a Kafka Connect :term:`source connector` that produces data generated by jfairy to a topic.
It also provides a :term:`sink connector` that reads data from a topic. This is typically used for product demonstrations
around Kafka Connect.
